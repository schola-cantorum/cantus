## 1. Ollama provider adapter（落實 Requirement「OllamaChatModel subclasses OpenAIChatModel and defaults to the local Ollama daemon」與 Requirement「OllamaChatModel surfaces an actionable error when the daemon is unreachable」、呼應 Decision「Ollama adapter subclasses OpenAIChatModel via Ollama's OpenAI-compatible endpoint」與「default api_key 用 sentinel "ollama" 字串、不走 resolve_api_key env 解析」與「`OllamaChatModel.chat()` / `.stream()` wrap `openai.APIConnectionError` 為 `ConnectionError`」）

- [x] 1.1 為 Requirement「OllamaChatModel subclasses OpenAIChatModel and defaults to the local Ollama daemon」建立 `cantus/model/providers/ollama.py`：宣告 module constants `OLLAMA_BASE_URL = "http://localhost:11434/v1"` 與 `OLLAMA_API_KEY_SENTINEL = "ollama"`；定義 `class OllamaChatModel(OpenAIChatModel)`，constructor 簽章 `__init__(self, model_id: str, api_key: str | None = None, base_url: str | None = None, **client_kwargs: Any)`，內部以 `resolved_key = api_key if api_key is not None else OLLAMA_API_KEY_SENTINEL`、`resolved_url = base_url if base_url is not None else OLLAMA_BASE_URL` 組值後呼 `super().__init__(model_id=model_id, api_key=resolved_key, base_url=resolved_url, **client_kwargs)`；**不**呼 `resolve_api_key`；驗證：`python -c "from cantus.model.providers.ollama import OllamaChatModel; m = OllamaChatModel('gemma3:4b'); print(m._api_key, m._base_url)"` 輸出 `ollama http://localhost:11434/v1`
- [x] 1.2 為 Requirement「OllamaChatModel surfaces an actionable error when the daemon is unreachable」在 `cantus/model/providers/ollama.py` 的 `OllamaChatModel` 內 override `chat(messages, tools=None, **kwargs)` 與 `stream(messages, tools=None, **kwargs)`：每個 method 內 `try: return super().<method>(...)`、`except openai.APIConnectionError as exc:` 後 `raise ConnectionError(f"Cannot reach Ollama daemon at {self._base_url}. Is \`ollama serve\` running? Install: https://ollama.com/download — then \`ollama pull <model>\` before retrying.") from exc`；`stream` override 要保持 generator 語意（用 `yield from super().stream(...)` 在 try 內）；不 catch 其它 exception；驗證：unit test `tests/providers/test_ollama_connection_error.py` 在 fake client `create` raise `openai.APIConnectionError` 場景下 assert `ConnectionError` message 含 4 條子字串（`Cannot reach Ollama daemon at` / `http://localhost:11434/v1` / `` `ollama serve` `` / `https://ollama.com/download`）
